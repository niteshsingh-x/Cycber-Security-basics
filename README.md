----------2 Mark Answers----------
1) Concept of Cyber Security
Cyber Security is the practice of protecting systems, networks, and data from cyber attacks.
It focuses on ensuring confidentiality, integrity, and availability of information.

2) Ethical Hacking
Ethical hacking is the process of legally testing systems and networks to find security
vulnerabilities.
It is performed by authorized professionals to improve security.

3) System Integrity Validation
System integrity validation is the process of ensuring that a system and its data are accurate,
consistent, and not tampered with.
It helps detect unauthorized changes.
